# ChatGPT-Interface

check versions:

- run `node -v`
- run `npm -v`

make client:

- run `npx create-react-app client`

make server:

- run `npm init`

start development:

- run `npm start`

express backend:

- run `npm install express cors body-parser morgan openai --save`

openai:

- generate api key
- install openai (done with express)
- use https://platform.openai.com/docs/api-reference/authentication
